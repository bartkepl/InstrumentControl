# R&S RTB2004 — 4-Channel Oscilloscope

## Overview

The Rohde & Schwarz RTB2004 is a 4-channel, 300 MHz digital oscilloscope with LAN and USB connectivity.

| Specification | Value |
|---|---|
| Type | Digital Oscilloscope |
| Channels | 4 analog + 16 digital (with option) |
| Bandwidth | 300 MHz |
| Interface | LAN (VXI-11), USB-TMC |
| Driver name | `RTB2004` |
| Resource patterns | `TCPIP?*::?*::INSTR`, `USB?*::?*::INSTR` |

---

## Connecting

1. Connect the RTB2004 via LAN or USB
2. Click **Add Instrument** and select the resource
3. Auto-detection parses `*IDN?` (`Rohde&Schwarz,RTB2004`)
4. Click **Connect**

For LAN connection, note the instrument's IP address from its **Utilities → Network** menu and configure it in NI-MAX or enter the TCPIP resource string manually: `TCPIP0::192.168.x.x::inst0::INSTR`

| Control | Description |
|---|---|

## Sequence Blocks

### RTB2004_SetChannel

Configures a single channel.

| Property | Type | Description |
|---|---|---|
| **Instrument** | InstrumentSelector | Connected RTB2004 instance |
| **Channel** | Combo | CH1, CH2, CH3, CH4 |
| **Enabled** | CheckBox | Enable or disable the channel |
| **V/div** | Number | Vertical scale in V/div |
| **Coupling** | Combo | DC, AC, GND |
| **Offset** | Number | Vertical offset in V |

### RTB2004_SetTimebase

Sets the horizontal timebase.

| Property | Type | Description |
|---|---|---|
| **Instrument** | InstrumentSelector | Connected RTB2004 instance |
| **Time/div (s)** | Number | Horizontal scale in seconds per division |

### RTB2004_SetTrigger

Configures the trigger.

| Property | Type | Description |
|---|---|---|
| **Instrument** | InstrumentSelector | Connected RTB2004 instance |
| **Source** | Combo | CH1–CH4, EXT |
| **Level (V)** | Number | Trigger threshold |
| **Slope** | Combo | POS (rising), NEG (falling) |

### RTB2004_Run

Starts continuous acquisition (`RUN`).

| Property | Type | Description |
|---|---|---|
| **Instrument** | InstrumentSelector | Connected RTB2004 instance |

### RTB2004_Stop

Stops acquisition (`STOP`).

| Property | Type | Description |
|---|---|---|
| **Instrument** | InstrumentSelector | Connected RTB2004 instance |

### RTB2004_Single

Arms the oscilloscope for a single-shot acquisition and waits until it completes.

| Property | Type | Description |
|---|---|---|
| **Instrument** | InstrumentSelector | Connected RTB2004 instance |
| **Timeout (ms)** | Number | Maximum wait time before the block fails |

### RTB2004_Autoscale

Sends `AUT` to auto-configure vertical and horizontal settings.

| Property | Type | Description |
|---|---|---|
| **Instrument** | InstrumentSelector | Connected RTB2004 instance |

### RTB2004_Measure

Reads an automated measurement from the oscilloscope.

| Property | Type | Description |
|---|---|---|
| **Instrument** | InstrumentSelector | Connected RTB2004 instance |
| **Channel** | Combo | CH1–CH4 |
| **Measurement** | Combo | FREQ, PER, AMPL, HIGH, LOW, MEAN, RMS, RISE, FALL, WIDT (pulse width) |
| **Variable** | Variable | Receives the measurement result |

### RTB2004_ReadWaveform

Downloads raw waveform data for one channel.

| Property | Type | Description |
|---|---|---|
| **Instrument** | InstrumentSelector | Connected RTB2004 instance |
| **Channel** | Combo | CH1–CH4 |
| **Variable Prefix** | Text | Variables `{prefix}_samples`, `{prefix}_dt`, `{prefix}_t0` are set |

## Example Sequence — Triggered Waveform Capture

```
[Start]
→ [RTB2004_SetChannel, CH=CH1, Vdiv=1.0, Coupling=DC]
→ [RTB2004_SetTimebase, Tdiv=0.001]
→ [RTB2004_SetTrigger, Source=CH1, Level=0.5, Slope=POS]
→ [RTB2004_Single, Timeout=5000]
→ [RTB2004_Measure, CH=CH1, Meas=FREQ, Variable=freq]
→ [LogMessage "Frequency: {freq} Hz"]
→ [SaveCsv, Path="C:\Data\scope_meas.csv"]
→ [End]
```
